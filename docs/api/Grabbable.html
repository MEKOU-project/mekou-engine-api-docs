Grabbable

オブジェクトの掴み（Grab）操作と空間内での移動を制御するコンポーネント

HOME THOUGHT API SAMPLES PROGRESS

Properties

  • isGrabbed: boolean
    オブジェクトが現在掴まれているかどうか。
  • position: Vector3
    オブジェクトの現在のワールド座標。

Methods

grab(handPosition: Vector3)

指定された手の座標でオブジェクトを掴みます。

release()

オブジェクトを離します（Grab状態の解除）。

updateGrabbedPosition(handPosition: Vector3)

掴まれている間、手の動きに合わせてオブジェクトの座標を更新します。

Events (Optional)

onGrab()

オブジェクトが掴まれた瞬間に呼び出されるコールバック。

onRelease()

オブジェクトが離された瞬間に呼び出されるコールバック。

Related Interfaces

Attachment

HUDスロットや他のGameObjectに自身を吸着・固定します。

  • attachTo(target, slotIndex): ターゲットに固定。
  • detach(): 固定を解除。

RaycastTarget

オブジェクトをレイキャスト（視線や指差し）の対象としてマークします。

  • setHighlight(enabled, color): ハイライト表示の制御。
  • isTargeted(): 現在ターゲットされているかの確認。
