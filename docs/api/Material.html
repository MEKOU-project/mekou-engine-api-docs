Material

オブジェクトの表面特性（色、不透明度）を定義・制御するコンポーネント

HOME THOUGHT API SAMPLES PROGRESS

Properties

  • color: { r: number, g: number, b: number }
    現在のマテリアルのRGBカラー値（0.0〜1.0）。

Methods

setColor(r: number, g: number, b: number)

マテリアルの色を更新します。リアルタイムな色の変化（ハイライト反応など）に使用されます。

setOpacity(opacity: number)

オブジェクトの不透明度を設定します。0.0（完全透明）から1.0（不透明）の範囲で指定可能です。

Usage Example

RaycastTargetと組み合わせることで、視線が合った瞬間に setColor() を呼び出し、ネオンのように発光させる演出などが可能です。
