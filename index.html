Welcome to MEKOU

「火焔の記憶、デジタルへの昇華」

MEKOU ENGINEは、古代日本の縄文文化が持つ力強い造形美と、現代のサイバーテクノロジーを融合させた実験的な開発基盤です。直感的なコンポーネントシステムにより、複雑なシステムを「縄」を編むように構築することが可能です。

Quick Start

$ git clone mekou-engine
$ cd mekou-engine
$ npm install && npm start
